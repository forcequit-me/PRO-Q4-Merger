Pro-Q4 Preset Merger

Merge multiple FabFilter Pro-Q 4 preset files (.ffp) into a single preset, reducing the number of plugin instances in your chain.

How to use

  • Add preset slots and load .ffp files via drag-and-drop or file picker
  • Click band dots on the EQ display to select/deselect bands for merging
  • Use the Overlay view to see all presets at once, or Tabs to view individually
  • Edit any band by clicking its dot — adjust frequency, gain, Q, shape, and more
  • Choose merge mode, configure output name, and click Merge & Download

Pro-Q 3 vs Pro-Q 4 Format

Pro-Q 3 uses a binary format (signature FQ3p, ~1348 bytes) which is not directly compatible. Pro-Q 4 uses a text-based INI format (signature FQ4p). If you have Pro-Q 3 presets, open them in Pro-Q 4 and re-save to convert.

Limitations

  • Maximum 24 bands per preset (Pro-Q 4 limit)
  • Exceeding 24 bands auto-splits into multiple output presets (A, B, C…)
  • Smart Merge suggests combining nearby bands — review each suggestion carefully

Band Inclusion

Active bands (Used=1) are always selected. Bands with Used=0 but non-default parameters are detected as "inactive but configured" and shown with reduced opacity — click to include them.
